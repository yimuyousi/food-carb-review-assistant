Copy this template to `.private/project_context.local.md` when a user needs project-specific writing support.

Do not commit `.private/project_context.local.md` to GitHub.

## Writing Preferences

- Prefer critical synthesis over paper-by-paper summaries.
- Use citation placeholders such as `[P001]`.
- Mark unsupported claims as `needs_manual_check`.
- Separate correlation from causality.
- Separate method transferability from direct evidence.
